# YiLa Web Control

这是一个浏览器端 YiLa BLE 开门项目。协议核心在 TypeScript 包里，网页端通过浏览器 Web Bluetooth 连接设备。

## 目录

```text
yila/
  packages/
    yila-core/      # 协议核心：MD5、AES-ECB、开门指令、响应解析
  apps/
    web/            # Web Bluetooth 网页版
```

## 运行

```bash
npm install
npm run dev
```

然后访问：

```text
http://192.168.1.155:5173/
```

也可以在本机访问：

```text
http://localhost:5173/
```

注意：局域网 HTTP 地址可以打开页面，但 Web Bluetooth 通常不会在非 HTTPS 的局域网地址下工作。需要实际连接蓝牙时，优先在部署机器上用 `http://localhost:5173/` 访问，或给局域网地址配置 HTTPS。

## 构建

```bash
npm run build
```

## 硬件接口

硬件协议、BLE UUID、开门指令、响应解析和扩展约定见 [docs/硬件接口文档.md](docs/硬件接口文档.md)。

## 国际化

Web 端支持中文和英文。中文环境会自动显示中文，也可以在页面右上角手动切换。

## 使用

没有设备时，主页面会直接显示搜索添加面板，在浏览器蓝牙弹窗里选择 YiLa 设备。已有设备时，通过“添加设备”入口展开搜索面板。添加成功后页面会显示设备卡片，卡片上可以开门或进入该设备设置；删除设备在设置页里操作。

## 管理后台

搜索名称前缀在搜索区域里。开锁时间、等待时间、关锁时间和反向开锁设置会跟随具体设备保存：点击设备卡片上的“设置”，保存该设备自己的参数。密码不会写入项目文件，网页运行时输入。
